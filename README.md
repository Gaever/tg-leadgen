# Telegram LeadGen

## Запуск

```bash
cp .env.example .env
# заполни .env

docker-compose up --build
```

- Frontend: http://localhost:3000
- API: http://localhost:8000/docs

## Использование

1. Авторизуйся в Telegram
2. Выбери чат → скачай сообщения  
3. Вкладка "Поиск" → RAG по скачанным сообщениям
